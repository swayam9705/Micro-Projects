# Multi-Tool Web Utility

A powerful, open-source web application providing a suite of image manipulation and data utility tools. From generating vibrant color palettes using K-Means clustering to transforming images into ASCII art, this platform leverages a high-performance FastAPI backend and a responsive React frontend.

* **Color Palette Generator:** Extract dominant color schemes from images using clustering.
* **ASCII Art Converter:** Transform standard image formats into stylistic ASCII representations.
* **Real-time Processing:** Fast, asynchronous image processing powered by FastAPI.
* **Clean UI:** A minimalist and intuitive interface built with React.

## ‚öôÔ∏è Getting Started

### Prerequisites

* Node.js (v16+)
* Python (3.9+)
* Pip / Virtualenv

### Installation

1. **Clone the Repository**
```bash
git clone https://github.com/swayam9705/Micro-Projects.git
cd your-repo-name

```


2. **Backend Setup**
```bash
cd backend
python -m venv venv
source venv/bin/activate  # On Windows: venv\Scripts\activate
pip install -r requirements.txt
uvicorn main:app --reload

```


3. **Frontend Setup**
```bash
cd ../frontend
npm install
npm run dev

```
